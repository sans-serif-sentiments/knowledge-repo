# Summary
Reusable pattern for coordinating messaging, enablement, and user-facing updates when shipping a major feature.

# Details
## Stages
1. **Narrative Draft**: align PM + PMM on value prop; reference concept docs.
2. **Proof Deck + FAQ**: synthesize enablement materials; embed metrics with citations.
3. **Internal Review**: run through Comms + Legal; verify policies CO-001 compliance.
4. **External Rollout**: publish product updates, email, in-app banners; log IDs in launch tracker.

## Checklist
- ✅ Metrics validated against MT-001 definitions.
- ✅ Enablement team trained via EN-FAQ-001.
- ✅ Support macros updated.
